# Input Validation

**`Config File Host Path`**
Accepted formats are:

- Valid filesystem paths (eg. `/mnt/path/to/config.yml`)

Regex used to match those: `^\/([a-zA-Z0-9._-]+(\s?[a-zA-Z0-9._-]+|\/?))+$`
You can try live [here](https://regex101.com/r/9gwRQK/1)
